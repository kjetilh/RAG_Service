# Answer Template - Bokskriving

Bruk denne strukturen nar sporsmalet handler om kapittelskriving, syntese eller argumentasjon.

1. **Kort hovedpoeng**
   En kort og presis oppsummering av hva materialet stotter.

2. **Hva litteraturen stotter**
   Oppsummer relevante poenger fra artikler og annen faglig dokumentasjon.

3. **Hva intervjuene tilfører**
   Oppsummer hva intervjuene bekrefter, nyanserer eller utfordrer.

4. **Syntese for boktekst**
   Foresla en skrivbar syntese med tydelig merking av hva som er direkte dokumentert og hva som er inferens.

5. **Mulig kapittel- eller avsnittsstruktur**
   Gi 3-6 konkrete punkter som kan brukes som disposisjon eller avsnitt.

6. **Hull eller svak dekning**
   Pek ut hva som mangler, er svakt dokumentert eller spriker.

7. **Kilder**
   List relevante kilder som [1], [2], osv.

Retningslinjer:
- Ikke skriv boktekst som later som den er ferdig hvis dekningen er svak.
- Ikke bland litteratur og intervjuer sammen uten a si hva som kommer fra hvor.
- Vaer eksplisitt hvis du gjør en redaksjonell syntese.
